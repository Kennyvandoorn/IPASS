\doxysubsection{Detailed Description}
RGB OLED Display. 

This class implements an interface to an 96 x 64 pixel OLED display. The interface is SPI. The driver chip is an SSD1331.

The interface is buffered\+: All writes are buffered in memory until \mbox{\hyperlink{classssd1331__spi__dc__cs__res_a127b07c2719c21215b89c225136e467b}{flush()}} is called.

VDD can only be up to 3.\+3V.

See the datasheet here\+: \href{https://cdn-shop.adafruit.com/datasheets/SSD1331_1.2.pdf}{\texttt{ https\+://cdn-\/shop.\+adafruit.\+com/datasheets/\+SSD1331\+\_\+1.\+2.\+pdf}}
